## Transactions And Multi-DB

These are handled specially in the connection loop rather than the per-db command registry:

- `SELECT`
- `FLUSHALL`
- `MULTI`
- `EXEC`
- `DISCARD`
- `WATCH`
- `UNWATCH`

## Known Omissions

The codebase contains a `DOCS` command module, but the server does not currently register it. Do not rely on `DOCS` being available over the network until the registry includes it.
